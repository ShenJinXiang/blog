java 命令行构建

习惯了使用eclipse、idea等集成开发环境以后，编译运行的基础命令都忘光光了，在此做个总结

建造者模式

建造者模式又称为生成器模式，是一种较为复杂、使用频率相对较低的创建信模式。建造者模式用于创建又多个部件组成的复杂产品

日期: 2017-06-08 | 分类: java | 标签: java 设计模式

策略模式

我们经常会遇到这样的情况，实现某一个功能有多种途径，每一种途径对应一种算法，此时我们可以使用一种设计模式来实现灵活地选择解决途径，也能够方便地增加新的解决途径。策略模式就是为了适应算法灵活性而产生的设计模式

日期: 2017-06-01 | 分类: java | 标签: java 设计模式

观察者模式

开发过程中会有这样的情况：一个对象发生改变时，需要影响其他一个或多个对象也发生改变，这时候需要用到观察者模式。观察者模式是使用频率最高的设计模式之一...

日期: 2017-05-26 | 分类: java | 标签: java 设计模式

命令模式

在软件开发中，我们经常需要向某些对象发送请求（调用其中的某个或某些方法），但是并不知道请求的接收者是谁，也不知道被请求的操作是哪个，此时，我们特别希望能够以一种松耦合的方式来设计软件...

日期: 2017-05-23 | 分类: java | 标签: java 设计模式

组合模式

树形结构在软件中随处可见，例如操作系统中的目录结构、应用软件中的菜单、办公系统中的公司组织结构等等。组合模式就是用来处理树形结构的一种模式

日期: 2017-05-21 | 分类: java | 标签: java 设计模式

抽象工厂模式

在工厂方法模式中具体工厂负责生产具体的产品，每一个具体工厂对应一种具体产品，工厂方法具有唯一性，一般情况下，一个具体工厂中只有一个或者一组重载的工厂方法。但是有时候我们希望一个工厂可以提供多个产品对象，而不是单一的产品对象

日期: 2017-05-19 | 分类: java | 标签: java 设计模式

模板方法模式

模板方法模式是结构最简单的行为型设计模式，在其结构中只存在父类与子类之间的继承关系

日期: 2017-05-14 | 分类: java | 标签: java 设计模式

适配器模式

适配器模式将一个接口转换成客户希望的另一个接口，使接口不兼容的那些类可以一起工作，其别名为包装器(Wrapper)

日期: 2017-05-10 | 分类: java | 标签: java 设计模式

外观模式

在开发过程中，有时为了完成一个复杂的功能，一个客户类需要与多个业务类进行交互，而这些需要交互的类经常做为一个整体出现，由于涉及到的类比较多，导致使用时代码复杂，这个时候可以使用外观模式
